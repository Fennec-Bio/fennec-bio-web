How does a user add or remove an experiment from the cohort?

In the existing sidebar you click a row to toggle. With two columns, there are a few common UX patterns.

A

Click row to toggle (consistent with current sidebar)

Click a candidate row → it moves into the cohort column (and a checkmark/highlight appears). Click a row in the cohort column → it leaves the cohort. No drag, no buttons. Bulk "All / None" buttons stay above each column.

Explicit "Add →" / "← Remove" buttons

Each candidate row has an "Add →" button on the right. Each cohort row has a "← Remove" or × button. Less ambiguous about what clicking does, but heavier visual weight.
